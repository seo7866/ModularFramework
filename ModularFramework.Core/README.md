# ModularFramework.Core

ModularFramework의 핵심 실행 기반을 담당하는 모듈입니다.

---

## 📌 개요

ModularFramework.Core는 프레임워크의 저수준 기반 기능을 제공합니다.

주요 역할은 다음과 같습니다:

- 어셈블리 스캔 (Assembly Scanning)
- 타입 로딩 및 탐색
- 의존성 분석
- 런타임에서 필요한 메타 정보 제공

---

## 🧱 역할

이 모듈은 애플리케이션의 실행 흐름을 직접 제어하지 않습니다.

대신, 다른 모듈들이 사용할 수 있는 기반 기능을 제공합니다.

---

## ⚙ 주요 기능

- 현재 실행 중인 어셈블리 탐색
- 참조된 의존 어셈블리 로딩
- 타입 검색 및 필터링
- 런타임 메타데이터 제공

---

## 💡 설계 방향

이 모듈은 다음 원칙을 따릅니다:

- 외부 프레임워크 의존성 최소화
- 순수한 로딩/탐색 책임만 수행
- DI, 설정, 매핑 로직을 포함하지 않음

---

## 📦 위치 역할

ModularFramework.Core는 다음 모듈들의 기반이 됩니다:

- DependencyInjection → 타입 탐색 결과 기반 등록
- Configuration → 타입 기반 설정 바인딩
- Data → Query 타입 분석
- WebViewKit → 런타임 로딩 기반 확장

---

## 🚧 상태

- Stable (기반 모듈)
- 변경 최소화 대상
